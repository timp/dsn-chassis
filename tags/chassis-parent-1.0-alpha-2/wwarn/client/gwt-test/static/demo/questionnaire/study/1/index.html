WWARN Study Questionnaire - Prototype 1

This page is a prototype of the study questionnaire for studies submitting data to WWARN.

The first section in the question is a set of general questions which studies applying to any WWARN module will be asked to complete.

After the general questions will be one section for each of the WWARN modules. Of these, only those sections that are relevant to the study will be shown to the user. However, all sections will be shown here for convenience and comparison.
